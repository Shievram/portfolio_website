# Shievram Portfolio (Next.js)

A Brittany-inspired portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Motion (installed and ready for future animation work)

## Project structure

- `src/app` - Next.js App Router pages and global styles
- `src/components` - reusable UI sections
- `src/data/portfolio.ts` - all portfolio content in one place

## What to customize

- Replace `metadataBase` and Open Graph URL in `src/app/layout.tsx`
- Update project cards in `src/data/portfolio.ts`
- Add screenshots or assets in `public/`
- Deploy to Vercel
